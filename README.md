# Financial-Analysis-Dashboard-Power-BI

•	Designed and developed an interactive Power BI dashboard to analysed income, expenses, savings, and targets.

•	Modelled multi-year financial data (2021–2024) with 10+ DAX measures and calculated tables, enabling detailed time-series and category-level analysis.

•	Built 10+ dynamic visuals (line charts, donut charts and tables) to track income, expenses, savings %, and year-wise performance.

•	Implemented time-based slicers, drill-downs, and filters to provide interactive and user-friendly reporting.
